### 1. Постановка задачи

Задача 1.
Проанализировать последовательность, данную в задание, найти закономерность и написать алгоритм.

Задача 2.
Рассмотреть все возможные подпоследовательности, которые начинаются с первого элемента последовательности. Найти максимальный и максимальный элементы подпоследовательностей, их разность. Вывести длину первой подпоследовательности, где разность максимальная.

Задача 3.
Проанализировать шаги бегуна, понять, попадают ли они в данные на входе точки. Вычислить номер круга, когда это произошло и номер шага.

Задача 4.
Проанализировать массив, состоящий из n чисел. Найти элемент, значение которого равно числу различных элементов в массиве.

### 2. Входные и выходные данные

#### Данные на вход

Задача 1.
На вход поступает число n - количество цифр в итоговой последовательности.

Задача 2.
На вход поступает число n и n целых чисел – элементы последовательности.

Задача 3.
На вход поступает целое число L и целые A, B, C (попарно различны, в диапазоне [0, L)), далее последовательность целых чисел – шаги бегуна; последний элемент ввода равен -L*L и не учитывается как шаг.

Задача 4.
На вход поступает число n - количество элементов в массиве и сами элементы последовательности.

#### Данные на выход

Задача 1.
На выходе остаётся последовательность из n-чисел

Задача 2.
На выходе остаётся длина префикса

Задача 3.
На выходе остаётся номер круга на этот момент и номер шага i

Задача 4.
На выход мы получим значение элемента последовательности, равному числу различных элементов в массиве (предполагается, что такой есть), а также индекс этого элемента

### 3. Алгоритм

#### Алгоритм выполнения программы задачи 1:

1. Ввод данных:  
   Программа считывает целое число - n.

2. Анализ решения:  
   Представим, что индексы начинаются с 1. Тогда на нечетных позициях будут идти степени двойки, где степень это индекс. На четных позициях факториал индекса. Достаточно создать цикл с индексами, начиная с 1, и заканчиваем (n+1), где нужно пройтись по четным и нечетным элементам, выполняя действия факториала и степени соответственно.

3. Вывод результата:  
   На экран выводится n чисел последовательности

#### Алгоритм выполнения программы задачи 2:

1. Ввод данных:
   Программа считывает число n и n целых чисел.

2. Анализ решения:
   Примем за максимальное и минимальные числа элемент с нулевым индексом (первый элемент). В цикле начнем перебирать подпоследовательности, начиная с минимальных. Меняем минимум и максимум, сравнивая их с элементами, вычисляем разность, фиксируем и сравниваем с дополнительной, заведомо отрицательной, переменной. Как только максимальная разность будет найдена, отдельная переменная зафиксирует длину подпоследовательности и выведет её.

3. Вывод результата:  
   На экран выводится длина подпоследовательности с максимальной разностью между максимальным и минимальным элементами.
   
#### Алгоритм выполнения программы задачи 3:

1. Ввод данных:
   Программа считывает целое число L и целые A, B, C, далее последовательность целых чисел до -L*L.
   
3. Анализ решения:
   Создадим цикл, который будет принимать числа до -L*L. Сразу запускаем счётчик шагов, счётчик суммы шагов и счётчик кругов. Далее будем проверять количество шагов на совпадение с A, B, C. Как только совпадение будет найдено, выведем номер круга на этот момент и номер шага, если такого не найдется, то выведем NO.

4. Вывод результата:  
   На экран выводится номер круга и номер шага i.
   
#### Алгоритм выполнения программы задачи 4:

1. Ввод данных:
   Программа считывает число - n - количество элементов в массиве. Далее сами элементы.

2. Анализ решения:
   Создаем цикл, чтобы записать все элементы в массив. Дополнительно создаю функцию, которая находит количество разных элементов массива. Находим количество разных элементов массива и сравниваем каждый элемент массива с ним, как только совпадение будет найдено, цикл прервется и выведет этот элемент и его индекс. 

4. Вывод результата:  
   На экран выводится значение элемента последовательности, равному числу различных элементов в массиве, а также индекс этого элемента.

import java.util.Arrays;
import java.util.Scanner;
public class лабораторная4 {
    public static void main(String[] args) {
        Scanner in = new Scanner(System.in);
        int k = -1;
        int n = in.nextInt();
        int []x = new int[n];
        for (int i = 0; i < x.length; i++) {
            x[i] = in.nextInt();
        }
        int s = f(x);
        int p = 0;
        for (int i = 0; i < x.length; i++) {
            if (x[i] == s) {
                p = x[i];
                k = i;
                break;
            }
        }
        System.out.println("Элемент последовательности, который равен числу различных элементов в массиве:" + " " + p);
        System.out.println("Индекс элемента в массиве:" + " " + k);
    }
    public static int f(int [] x) {
        if (x.length == 0)
            return 0;
        Arrays.sort(x);
        int c = 1;
        for (int i = 1; i < x.length; i++) {
            if (x[i] != x[i - 1])
                c++;
        }
        return c;
    }
}
